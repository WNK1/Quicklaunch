# QuickLaunch v2 — Start Menu Panel

Панель запуска приложений, встроенная в область Start Menu Windows 11.
Открывается по нажатию **Win+Q** — точно на том же месте где появляется Start Menu.

---

## Установка

### 1. Node.js
→ https://nodejs.org (LTS)

### 2. Запусти SETUP.bat
Установит зависимости и запустит приложение.

### 3. Готово
- Нажми **Win+Q** → появится панель
- Нажми ещё раз или **Esc** → скроется
- Иконка в трее: правый клик для меню

---

## Управление

| Действие | Результат |
|---|---|
| **Win+Q** | Показать / скрыть панель |
| **Esc** | Скрыть панель |
| Клик вне окна | Скрыть панель |
| ПКМ по трею | Меню: показать, выход |

---

## Автозапуск

Включи переключатель **«Автозапуск»** прямо в панели —
запишет путь в реестр `HKCU\...\Run`.

---

## Логика запуска приложений

| Значок | Смысл | Действие |
|---|---|---|
| 🟢 зелёная точка | Приложение найдено на ПК | Запускает .exe |
| 🟡 оранжевая точка | Не найдено | Открывает сайт загрузки |

---

## Добавить своё приложение

**main.js** → `APP_DEFS`:
```js
myapp: {
  exePaths: ['C:\\Program Files\\MyApp\\myapp.exe'],
  website: 'https://myapp.com/download',
},
```

**src/index.html** → в нужный раздел `CATS`:
```js
{ key:'myapp', n:'My App', e:'🟠', c:'rgba(255,100,0,0.15)' },
```

npm run build
